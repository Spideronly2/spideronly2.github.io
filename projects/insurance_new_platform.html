Stock Dashboard and Management

  • Frameworks: Spring Boot
  • Languages: Java, Golang, Typescript, Python, HTML, CSS
  • Tech Stacks: Rest, MongoDB, GCP Cloud Run, GCP Kubernetes, GCP Scheduler, Stripe Subscriptions, Auth0, Docker, Cucumber.js, Cucumber/JVM

Project goal: The customer is using a non-flexible platform using old technologies which is hard to maintain and provides bad user experience. They need to improve the user experience and time to market in order to achieve their business goals.

We implemented a new platform using new technologies that provides faster deployment, better development maintenance, and better user experience.This platform was implemented successfully in phases from their new business products in different countries. The solution was implemented using the following stack: spring boot, microservices, auth0, GCP kubernetes, mongoDB, stripe. I developed many spring boot modules like security, caching, actuator and many integrations with stripe, and thirdly parties such as MID UK providers, google doc, docmosis.On daily basis using intellij, mockito, bdd cucumber (jvm/Javascript), karate and deployment with Github. I used vue.js for oauth0 test. All tests running as docker and production in kubernetes cluster.
